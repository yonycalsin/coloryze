## Getting Started

### Install

Create the project.

```bash
git clone https://github.com/yonycalsin/coloryze
```

Access the project directory.

```bash
cd coloryze
```

Install dependencies.

```bash
npm ci
```

Serve with hot reload at http://localhost:3000.

```bash
npm run dev
```

### Lint

```bash
npm run lint
```

### Build

```bash
npm run build
```

### Test

```bash
npm run test
```
